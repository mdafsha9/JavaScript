Javascript Variables

There are 3 ways to declare a Javascript variable

  • Using var
  • Using let
  • Using const

Javascript identifiers

  • All javascript varibles are identified with unique names
  • This unique names are called identifiers
  • identifiers can be short names like x & y or more descriptive like (name, age)
  • Rules for constructing names for variable are
  • Javascript variable name can contain letters, digits, underscore and dollarsign
  • A javascript variable must begin with a letter/underscore(_)/dollarsign($)
  • Javascript variables are case-sensitive (y & Y are two different variables)
  • reserve words (like javascript keywords) cannot be used as names
