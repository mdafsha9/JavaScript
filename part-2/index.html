Functions

defination

A JavaScript function is a block of code designed to perform a particular task. A JavaScript function is executed when something invokes it.

  • A function is defined with a function keyword followed by function name followed by paranthesis
  • A function name can contains letters, digits, underscore(_) and dollarsign($). (same rule as variables)
  • A function parameters are listed inside the paranthesis in the function defination
  • A function arguments are the values recivied by the function, when it is invoked.
  • Inside the function, the function arguments, the function parameters are treated as local variables

Function Invocation

  • Javascript code is executed when something invokes it
  • When an event occurs (user click, hover)
  • When it is invoked from Javascript code
  • Automatically(self-invoked)

Why functions

  • we can reuse code: Define the code once, use it many times
  • Define the code once and use it many times with different arguments, to produce different results
